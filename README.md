# ResumeIQ AI

ResumeIQ AI is a full-stack application designed to analyze, score, and optimize resumes using AI.

## Project Structure

```text
resumeiq-ai/
│
├── frontend/     ← Store ALL React.js frontend files here
│
├── backend/      ← Store ALL Node.js, Express.js, MongoDB backend files here
│
├── README.md
└── .gitignore
```

## Running the Application

### Frontend
1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

### Backend
1. Navigate to `backend/`
2. Install dependencies: `npm install`
3. Start server: `npm run dev` or `npm start`
